Välkommen till ppl-pocket¶

Pocketboken innehåll styrs efter kunskapskraven från Transportstyrelsen Version 1, 2010-02-02.

Warning

PPL-pocket är för närvarande i väldigt tidigt stadie men kommer successivt att fyllas på.

Om du hittar felaktigheter så kan du antingen bidra genom Github eller mail mig carlodin@gmail.com.

Huvudmål:

  • Kort och koncist
  • Tydligt vad som krävs för ppl nivå

PPL-pocket är open source och licensierat under Creative Commons. Om man vill bidra så kan man göra så via github.com/loa/ppl-pocket.
